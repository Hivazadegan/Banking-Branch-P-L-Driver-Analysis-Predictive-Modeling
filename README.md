# Banking Branch P&L Driver Analysis & Predictive Modeling

## 🚀 Overview
This repository outlines the methodology for identifying key Profit & Loss (P&L) drivers across 1,800+ bank branches. The project leverages machine learning to transition from reactive reporting to proactive operational strategies.

## 🔑 Key Features
- **Data Engineering:** Automated ETL pipeline for consolidating large-scale balance sheet datasets.
- **Predictive Analytics:** Implemented **Random Forest** and **Decision Trees** to pinpoint performance drivers.
- **Decision Support:** Developed actionable exit strategies for underperforming branches.
- **BI Excellence:** Advanced **DAX** implementation in Power BI for multi-dimensional financial monitoring.

## 🛠 Tech Stack
- **Languages:** Python (Pandas, Scikit-Learn)
- **Business Intelligence:** Power BI (DAX, Data Modeling)
- **Data Processing:** Excel (Advanced automation)
